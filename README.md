# Adaptive IDS ML Framework

Ongoing research-oriented machine learning project focused on intrusion detection systems (IDS), network security, and reducing false positive rates using traditional machine learning approaches.

## Project Status

🚧 Currently under development.

This repository is being developed as part of ongoing research preparation in the area of adaptive machine learning-based intrusion detection systems for cybersecurity applications.

## Research Focus

The project aims to explore machine learning approaches for detecting malicious network traffic while improving intrusion detection reliability and minimizing false positive rates.

## Planned Dataset

Primary Dataset:
- CICIDS2017 (MachineLearningCSV version)

Secondary Dataset:
- UNSW-NB15

## Planned Models

- Random Forest
- Support Vector Machine (SVM)
- XGBoost

1. Data preprocessing

4. Feature engineering
5. Model training
6. Performance evaluation
